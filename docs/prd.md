# 30秒直觉 MBTI 速测需求文档

## 1. 应用概述

### 1.1 应用名称
30秒直觉 MBTI 速测

### 1.2 应用描述
一款微信小程序，通过12道无需思考的直觉联想题，在30秒内快速帮助用户定位MBTI人格类型，并提供人性化的人格分析解读。

## 2. 核心功能

### 2.1 测量题目模块
- 提供12道直觉联想题，每题限时3秒反应
- 每题仅提供A/B两个选项供用户选择
- 采用流畅的交互动效：点击选项后，当前卡片向左滑出，新卡片从右侧淡入
- 题目内容详见附件中的题目设计清单

### 2.2 逻辑计算模块
- 统计用户在12道题中的选项分布
- 每3题对应一个维度（E/I、S/N、T/F、J/P）
- 在该维度的3题中，选择A或B较多的一方即为该人格代码
- 若某维度出现平衡（直觉不明显），则随机指定一个，并在分析中加入“复杂且多面”的描述词

### 2.3 结果分析页面
- 测量完成后，点击提交按钮跳转至人格分析页面
- 页面包含以下模块：
  - 模块A：顶部标语（格式：[四个字母代码]·[文艺化称呼]）
  - 模块B：人性化深度解析（300字左右的感性文字，温暖笃定的语气，像老友低语）
  - 模块C：社交分享卡片

### 2.4 分享功能
- 提供生成美图分享按钮
- 提供再测一次按钮
- 社交分享卡片包含：核心关键词、灵魂共鸣（一句话处世哲学）

## 3. 视觉规范

### 3.1 整体风格
遵循iOS 26 / 极简毛玻璃风格

### 3.2 底色
- 纯白色背景
- 点缀2-3个淡色系（马卡龙粉/蓝）的流体渐变色块
- 色块保持缓慢位移效果

### 3.3 容器样式
```
Background: rgba(255, 255, 255, 0.5)
Backdrop-filter: blur(25px) saturate(180%)
Border: 1px solid rgba(255, 255, 255, 0.3)
Border-radius: 32px
```

### 3.4 字体
- 默认系统字体（San Francisco / 萍方）
- 字间距 +0.05em

### 3.5 交互动效
- 点击选项后，当前卡片向左滑出
- 新卡片从右侧淡入
- 使用丝滑转场效果

## 5. 分析页面模块

### 5.1 顶部标语格式
[四个字母代码]·[文艺化称呼]

### 5.2 人性化深度解析要求
- 300字左右的感性文字
- 温暖笃定的语气
- 像老友在耳边低语
- 避免使用“该人格倾向于...”等描述
- 使用“你总是会在深夜里...”、“你的内心深处藏着一个...”等表达方式

### 5.3 社交分享卡片内容
- 核心关键词（如：逻辑的信徒、温柔的捕手）
- 灵魂共鸣（一句话形容你的处世哲学）
- 生成美图分享按钮
- 再测一次按钮
